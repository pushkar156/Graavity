# 🏥 Graavity Hospital Website

A modern, responsive, and professional healthcare website for **Graavity Multi-Speciality Hospital Pvt. Ltd. – Critical Care & Trauma Centre**, Nashik, Maharashtra.

Designed to establish a strong digital presence, build patient trust, showcase medical services and facilities, and provide easy access to hospital information.

---

## 🌟 Features

- Modern Healthcare UI/UX
- Fully Responsive Design
- Professional Hero Section
- About Hospital Section
- Medical Specialities Showcase
- Facilities & Infrastructure Display
- Doctor Profiles
- Patient Testimonials
- Photo Gallery
- Contact Information
- Google Maps Integration
- Emergency Contact CTA
- WhatsApp Quick Contact Button
- SEO-Friendly Structure
